About this site

How site was built to improve findability

The search engine optimization (SEO) best practices were followed to improve the findability of this website as described in the page SEO Best Practices in this website.

What i've been doing for increase Findability

  • Use Sematic HTML
  • Key phrase and key words have been included in h1, h2, h3... headings
  • HTML has been validated using W3validator
  • CSS has been validated using W3 Jigsaw Validator
  • The website has been validated with WAVE accessibility
  • The site has made to be responsive
  • The images have been optimized
  • The alt description has been added for each image
  • Research keyword to make website more findable
  • meta description has been added in each page with key words/ phrases
  • Keywords and key phrases rich content has been written on each page
  • hcard has been added with contact details
  • Integrated with Google Search Engine Console
  • site map xml is integrated with Google Search Engine Console
  • The website has been linked with social media such as Instagram and Facebook

How traffic will be analysed to discover findability problems

The traffic data/reports available from Google Search Engine Console (previously called "webmaster tools"), Awstats, and Webalizer from cpanel to answers questions like:

  • Using Google Webmaster or Bing Webmaster to check errors and perfomance of the website, url inspection, mobile usability and crawl stats
  • Using Awstats from cPanel, We can see visitors from which countries, what browser the visitors use and what is the visitors IP address
  • Is website effective? Using Awstats we can see the statistic of how long the visitors explore the website
